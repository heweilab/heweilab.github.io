哈希查找（Hash Search）

这是一种非常高效的数据检索技术，其核心思想是“通过关键字直接访问记录”，从而实现近乎常数级 O(1) 的平均查找时间。

一、核心思想与比喻

你可以把它想象成一个超级高效的图书馆管理系统：

  1. 传统方法（顺序或二分查找）：就像你进入一个按书名拼音排序的书架区，从A到Z一本本找，或者从中间开始二分查找。这需要时间。
  2. 哈希查找方法：图书馆有一个神奇的智能系统。你告诉它书名《三体》，它瞬间计算出一个编号 S253，然后直接让机器人去S253这个精确的书架位置把书取来。这个“计算编号”和“直接取书”的过程就是哈希查找。

这个比喻包含了哈希查找的三个最关键部分：

  • 关键字（Key）：书名《三体》。
  • 哈希函数（Hash Function）：那个神奇的“计算编号”的算法。
  • 哈希表（Hash Table）：所有书架位置的集合。

二、基本原理和步骤

哈希查找基于哈希表（Hash Table）这种数据结构。其工作流程如下：

  1. 存储数据（预处理）
    • 定义一个哈希函数 H(key)。
    • 对于每个要存储的数据，有一个唯一标识它的关键字（Key）（如学生的学号、商品的ID）。
    • 通过哈希函数计算关键字的哈希值：address = H(key)。这个值就是该数据在哈希表中理想的存储位置（索引）。
    • 将数据存储到哈希表的 address 位置。
  2. 查找数据
    • 给定要查找的关键字 key_x。
    • 用同样的哈希函数 H 计算其哈希地址：address_x = H(key_x)。
    • 直接去哈希表的 address_x 位置查看：
      • 如果该位置有数据，并且数据的关键字等于 key_x -> 查找成功！
      • 如果该位置是空的 -> 说明查找的数据不存在。
      • 如果该位置有数据，但关键字不匹配 -> 发生了“冲突”，需要按照预设的冲突解决方法继续查找（这是哈希查找的关键难点）。

字典数据类型 dict 的使用：把目标内容 target 当做字典的 Key ，要寻找的目标视为 Key 对应的 value ，即 return value=dict[target] if target in dict 。

注意：

字典的 Key 需要唯一，也就需要你寻找的目标解 target 是唯一解。

做完第一题，总结的以上内容，但是做完第二第三题发现之前的理解还是片面，哈希查找在应用当中还是很灵活。

1. 两数之和

已解答 简单

给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 target 的那 两个 整数，并返回它们的数组下标。

你可以假设每种输入只会对应一个答案，并且你不能使用两次相同的元素。

你可以按任意顺序返回答案。

示例 1：

输入：nums = [2,7,11,15], target = 9
输出：[0,1]
解释：因为 nums[0] + nums[1] == 9 ，返回 [0, 1] 。

示例 2：

输入：nums = [3,2,4], target = 6
输出：[1,2]

示例 3：

输入：nums = [3,3], target = 6
输出：[0,1]

提示：

  • 2 <= nums.length <= 104
  • -109 <= nums[i] <= 109
  • -109 <= target <= 109
  • 只会存在一个有效答案

解题代码：

暴力求解的方法就不列了，双循环遍历，只有一个解，找到了 return 即可

class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        dic = collections.defaultdict()
        for i in range(len(nums)):
            if target - nums[i] in dic:
                return dic[target - nums[i]],i
            dic[nums[i]] = i

dic = collections.defaultdict() 这行代码创建了一个特殊的字典对象，使用 collections.defaultdict 类型，它与普通字典的区别在于：当访问不存在的键时，不会抛出 KeyError 异常，而是返回一个默认值，括号里为指定默认值的类型。

哈希查找的字典创建，都应用 collections.defaultdict() 进行初始化。

49. 字母异位词分组

已解答 中等

给你一个字符串数组，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。

示例 1:

输入: strs = [“eat”, “tea”, “tan”, “ate”, “nat”, “bat”]

输出: [[“bat”],[“nat”,“tan”],[“ate”,“eat”,“tea”]]

解释：

  • 在 strs 中没有字符串可以通过重新排列来形成 "bat"。
  • 字符串 "nat" 和 "tan" 是字母异位词，因为它们可以重新排列以形成彼此。
  • 字符串 "ate" ，"eat" 和 "tea" 是字母异位词，因为它们可以重新排列以形成彼此。

示例 2:

输入: strs = [""]

输出: [[""]]

示例 3:

输入: strs = [“a”]

输出: [[“a”]]

提示：

  • 1 <= strs.length <= 104
  • 0 <= strs[i].length <= 100
  • strs[i] 仅包含小写字母

解题代码：

class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        dic = collections.defaultdict(list)
        for st in strs:
            key = "".join(sorted(st))
            dic[key].append(st)
        return list(dic.values())

如果使用 collections.defaultdict(list) 创建字典类型数据，可以直接使用 .append 对 list 进行赋值，当然如果不记得这个代码，可以创建一个空 list 来进行字典 value 的赋值，代码如下：

class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        dic = {}
        lis = []
        for st in strs:
            key = "".join(sorted(st))
            if key in dic:
                lis = dic[key]
            lis.append(st)
            dic[key] = lis
            lis = []
        return list(dic.values())

128. 最长连续序列

已解答 中等

给定一个未排序的整数数组 nums ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。

请你设计并实现时间复杂度为 O(n) 的算法解决此问题。

示例 1：

输入：nums = [100,4,200,1,3,2]
输出：4
解释：最长数字连续序列是 [1, 2, 3, 4]。它的长度为 4。

示例 2：

输入：nums = [0,3,7,2,5,8,4,6,0,1]
输出：9

示例 3：

输入：nums = [1,0,1,2]
输出：3

提示：

  • 0 <= nums.length <= 105
  • -109 <= nums[i] <= 109

解题代码：

我的一开始思路：暴力枚举求解。找到数组当中的序列当中最小的那个数，也就是 num-1 not in nums ，从这个数开始往上找，直到 num+i not in nums 这时关于这个最小数的最长序列长度就是 i ，保留整个数组的最大长度即为题目所求。另外题目限制了时间复杂度，这个时间复杂度是超的，为了通过测试，利用了 set() 集合的属性进行去重后求解。

class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        result = 0
        nums = set(nums)
        if len(nums) == 1:
            return 1
        for num in nums:
            if num-1 in nums:
                continue
            for i in range(1,len(nums)):
                if num+i not in nums:
                    if i>result:
                        result = i
                    break
                else: 
                    if i+1>result:
                        result = i+1
        return result

在Python中，set是一种无序且不重复的元素集合。它内部使用哈希表（hash table）实现，因此通常被称为哈希集合。所以虽然我用了两次循环，但是由于判断了num-1 not in nums 加上set() 的使用，相当于在第一次循环我跳过了非开头数据，第二次循环使用了 O(1) 的复杂度去寻找跳过的非开头数据，整体的时间复杂度在 O(N) ，满足了题目的要求。
